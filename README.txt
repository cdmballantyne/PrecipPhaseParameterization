#This repository contains the R objects for both the logistic regression and random forest version of
#Charlie Ballantyne's phase partitioning algorithm

The logistic regression model is stored in phaseclass_logreg.RData
The random forest model is story in phaseclass_rf.RData

More detail on these models can be found in http://hdl.handle.net/10012/19800

Citation:

Charlie Ballantyne (2023). Improving Predictions of Precipitation Phase over Canada with the Development of a Spatially Aware Parameterization of Rain/Snow Partitioning. UWSpace. http://hdl.handle.net/10012/19800
